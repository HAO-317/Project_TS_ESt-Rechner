# Overview

Project_TS_ESt-Rechner is a TypeScript-based application designed to calculate German income tax (Einkommensteuer, ESt) based on user-provided income data. The project leverages modern TypeScript practices to deliver a robust and maintainable solution for tax calculations, adhering to German tax regulations and formulas.

# Features

Accurate Tax Calculations: Computes German income tax based on the latest tax brackets and rules

User-Friendly Interface: Provides an intuitive interface for entering income details and viewing results.

TypeScript Implementation: Utilizes TypeScript for type safety and enhanced code maintainability.

Modular Design: Organized codebase with reusable components for easy updates and scalability.
